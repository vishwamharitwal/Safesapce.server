## Bug Analysis

### ThoughtsScreen Bugs (thoughts_screen.dart)

**Bug 1: Supabase channel not properly removed on dispose**
- Line 92: `_thoughtsSubscription?.unsubscribe()` only unsubscribes but doesn't remove the channel from Supabase client
- Fix: Use `_supabase.removeChannel(_thoughtsSubscription!)` for proper cleanup

**Bug 2: Timer callback race condition**
- Line 33-36: `_pollingTimer` fires `_fetchThoughts(isSilent: true)` which checks `mounted` internally, but the timer itself could fire during/after dispose in edge cases
- Fix: Add a `_isDisposed` flag and check it in the timer callback

**Bug 3: Realtime insert callback missing mounted check**
- Line 67-69: `_setupRealtimeSubscription` INSERT callback calls `_fetchThoughts(isSilent: true)` without checking `mounted` first
- Fix: Add `if (mounted)` guard before calling `_fetchThoughts`

**Bug 4: _CommentSheet FutureBuilder doesn't re-fetch after posting**
- Line 1188: After posting comment, `setState(() {})` rebuilds the widget but FutureBuilder won't re-run because the Future is created inline in build method
- Fix: Move Future to a method, use a `Key` or state variable to trigger refresh

**Bug 5: _CommentSheet FutureBuilder has no error handling**
- Line 1269-1271: Only checks `!snapshot.hasData`, doesn't check `snapshot.hasError`
- Fix: Add error state UI with retry button

### MainLayoutScreen Bugs (main_layout_screen.dart)

**Bug 6: SignalingService callbacks not cleared in dispose**
- Lines 47-56, 58-82: `onIncomingCall` and `onMatchFoundMain` are set but never cleared in dispose
- Fix: Set callbacks to null in dispose() method

**Bug 7: _showStyledNotification doesn't check mounted**
- Line 328-418: Uses `ScaffoldMessenger.of(context)` without checking `mounted`
- Fix: Add `if (!mounted) return;` at the start

**Bug 8: Timer callback doesn't check mounted before async**
- Line 248-249: Timer callback checks `if (mounted)` but only for the re-fetch, not for the timer existence
- Fix: Already handled by `_unreadDebounce?.cancel()` in dispose, but add `if (mounted)` guard at start of timer callback
